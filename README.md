# Font Info

Reads font information from file.

```ts
interface FontInfo {
  family: string;
  familylang: string;
  style: string;
  stylelang: string;
  fullname: string;
  fullnamelang: string;
  file: string;
  outline: boolean;
  scalable: boolean;
  lang: string;
  capability: string;
  fontformat: string;
  decorative: boolean;
  postscriptname: string;
  color: boolean;
  symbol: boolean;
  variable: boolean;
}
```

## Requirements

This package depends on `fc-scan`. Make sure you have it installed in your system.

## Installation

```bash
npm i get-font-info
```

## Usage

Using typescript or modern javascript:

```ts
import { getFontInfo } from "get-font-info";

const font = await getFontInfo("./Inter-Regular.otf");
console.log({ font });
```


Using es5:

```ts
const { geFontInfo } = require("get-font-info");

const font = await getFontInfo("./Inter-Regular.otf");
console.log({ font });
```
